#  **`drx[?]`** Show all debug registers


```text
Usage: drx Hardware breakpoints commands
```


- **`drx`** List all (x86?) hardware breakpoints
- **`drx <number> <address> <length> <perms>`** Modify hardware breakpoint
- **`drx-<number>`** Clear hardware breakpoint
